# Virtual Try-On (Starter)

This repo is a starter for the Virtual Try-On project.

Folders:
- src/api : pose, cloth, overlay modules
- src/app.py : Flask web app
- templates/ : HTML templates
- data/ : sample inputs
- static/ : outputs

Run locally:
1. pip install -r requirements.txt
2. python src/app.py
3. Open http://127.0.0.1:5000
